CLONAR REPOSITORIO LOCAL A GITHUB:
Creamos un nuevo repositorio en https://github.com.
Le damos nombre, descripción, seleccionamos si va a ser un proyecto publico o privado,
y dejamos el check de crear README sin marcar. Le damos a crear repositorio
y con esto ya tenemos el repositorio donde alojaremos nuestro proyecto local.

Luego ejecutar en visualStudio:

git init

git add .

git commit -m "first commit"

git remote add origin https://github.com/NOMBRE_USUARIO/NOMBRE_PROYECTO.git

git push -u origin master

git clone https://github.com/Luis-iom/pruebasgit.git (url del repositorio remoto)
git status (te dice si hay cambios en tus archivos,comparado al repo local)
git pull (trae todo lo nuevo del repo remoto al local)
git add . (agrega todo lo nuevo que agregue al repositorio local incluye lo borrado)
git commit -m "agregar comentario" (versiona los cambios agregados en el repo local)
git push origin "nombredelbranch"(los cambios en el repositorio local se van al remoto main en github)
git restore "nombredelarchivo.txt por ejemplo" (restaura a como esta el repositorio local)
git fetch (restaura localmente lo que está en el repositorio remoto)
git remote update (sólo para saber si el remoto cambió)
git checkout "nombredelbranch" (así te cambias de branch)
